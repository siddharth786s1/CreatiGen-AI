# CreatiGen-AI

## Overview
CreatiGen-AI is a cutting-edge project designed to leverage artificial intelligence for creative generation tasks. This tool aims to simplify and enhance the creative process for users by providing smart suggestions and automations.

## Features
- **AI-Powered Suggestions**: Generate creative ideas based on user input.
- **User-Friendly Interface**: Intuitive design that makes creativity accessible to everyone.
- **Customizable Templates**: Variety of templates to cater to different creative needs.
- **Real-Time Collaboration**: Work with others in real-time to brainstorm and develop ideas.

## Tech Stack
- **Front-end**: React, CSS
- **Back-end**: Node.js, Express
- **Database**: MongoDB
- **AI Library**: TensorFlow.js

## Installation Guide
To get started with CreatiGen-AI, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/siddharth786s1/CreatiGen-AI.git
   ```
   
2. Navigate into the project directory:
   ```bash
   cd CreatiGen-AI
   ```

3. Install the necessary dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm start
   ```

5. Open your browser and navigate to `http://localhost:3000` to see the application in action.
